## Description

CRUD Service to get some hands on NestJS, there is no business logic, or well structured code.  
This service is just a technical playground.  

## Stuff that was played with  

- [x] Model domains/entities
- [x] Add validation to entities
- [x] Add database support
- [ ] Add e2e test for validation
- [x] No business logic

## Installation

```bash
$ npm install
```

## Running the app

You need to have docker installed  
```bash
docker compose up -d
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
